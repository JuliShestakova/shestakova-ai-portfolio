# [shestakova-ai-portfolio](https://github.com/JuliShestakova/shestakova-ai-portfolio)

## 🚀 AI-Powered Frontend Portfolio: Neural HUD Showcase
This repository is a curated showcase of advanced UI components and AI-integration patterns extracted from the **KlikUm** platform. It demonstrates a high-end "Neural HUD" design language, immersive glassmorphism, and complex AI orchestration logic.

### 🛠️ Core Tech Stack
*   **Framework**: Next.js 16 (App Router)
*   **Styling**: Tailwind CSS 4 + Framer Motion
*   **AI Integration**: Vercel AI SDK (Streaming, Agent Rotation)
*   **Infrastructure**: Supabase SSR ready

### ✨ Key Showcases
*   **Neural HUD Layout**: A fully interactive 3D dashboard with scanline effects and immersive textures.
*   **Neural Chat UI**: A high-end chat interface demonstrating streaming-ready animations and AI-status orchestration.
*   **LiquidGlass Engine**: Advanced SVG-refraction engine for ultra-realistic glass effects.
*   **AI Agent Pool**: Logic for orchestrating Russian (Yandex/GigaChat) and International (OpenAI/Anthropic) AI models.
*   **NeuronOrb**: Performance-optimized canvas/motion animations for neural network visualization.
