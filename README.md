# [shestakova-ai-portfolio](https://github.com/JuliShestakova/shestakova-ai-portfolio)

## 🚀 AI-Powered Frontend Portfolio: Neural HUD Showcase
This repository is a curated showcase of advanced UI components and AI-integration patterns extracted from the **KlikUm** platform. It demonstrates a high-end "Neural HUD" design language, immersive glassmorphism, and complex AI orchestration logic.

### 🛠️ Core Tech Stack
*   **Framework**: Next.js 16 (App Router)
*   **Styling**: Tailwind CSS 4 + Framer Motion
*   **AI Integration**: Vercel AI SDK (Streaming, Agent Rotation)
*   **Infrastructure**: Supabase SSR ready

### ✨ Key Showcases
*   **LiquidGlass**: An advanced SVG-refraction engine for ultra-realistic glass effects.
*   **Neural HUD Layout**: Immersive, gamified grid systems with scanline effects.
*   **AI Agent Pool**: Advanced logic for orchestrating Russian (Yandex/GigaChat) and International (OpenAI/Anthropic) AI models.
*   **NeuronOrb**: High-performance canvas-based animations for neural network visualization.

Developed using **Antigravity** and **Cursor** AI-assisted tools for maximum engineering efficiency.
